Checklist mínimo

Datos a pedir a CCU

  • 3 a 5 usuarios piloto con área y rol.
  • CUITs agente/sociedades del grupo.
  • Lote de 20 a 50 proveedores reales.
  • Padrones/exportaciones disponibles.
  • Fuentes con acceso delegado o usuario técnico.
  • Criterios internos de aprobación/bloqueo.
  • Formato requerido de evidencia: Excel, PDF o link.

Día 0 · Preparación

  1. Confirmar usuarios y roles.
  2. Confirmar URL y Basic Auth.
  3. Completar Configuración.
  4. Cargar padrones disponibles.
  5. Registrar accesos.
  6. Ejecutar lote interno.

Día 1 · Demo guiada

  1. Mostrar Dashboard.
  2. Validar proveedor aprobable.
  3. Validar observado/bloqueado.
  4. Mostrar cola asistida.
  5. Mostrar legajo, Excel y PDF.

Semana 1 · Piloto

  1. CCU carga lote real.
  2. Impuestos revisa observados.
  3. Se cierran tareas asistidas.
  4. Se documentan falsos positivos/negativos.
  5. Se ajustan fuentes y criterios.
